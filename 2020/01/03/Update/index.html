UPDATE

Hi! I’m back! I took a pause from the blog. My thought was to start in 2020, so it’s about time i start!
What have i been doing? Well I’m on Christmas Break right now from school and i’ve been looking a lot more into C++. Both for Embedded and not Embedded. I’m still working on my 2D Game Engine but it’s on hold right now, last thing i did was fixing user input.

Moving on, i dived into C# Which i haven’t touched in a few years. Did some basic stuff and hopped into the Unity Game Engine and fibbled around with it while making a few simple games.

Since im a fan of F1, and like to play F1 2020 so I decided to create a UDP Telemetry app in C! Haven’t gotten it up on Github while writing this but will soon! It was quite interesting to work with packets and trying to recieve data from the game. Took some reading and to get the package right! I also tried another route with using Node.JS and Javascript to display it on a local server. So i could understand the data i was receiving at first. It was pretty cool!

I’m trying to use my spare time to combine hobbies and coding into learning. Like coding games and what not. I’ll keep this blog updated on what I’m doing, what I’m learning, and whats happening.
